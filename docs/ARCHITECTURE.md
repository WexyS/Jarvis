## Data Flow

1. **User Input** вҶ’ Orchestrator parses intent
2. **Task Router** decomposes into subtasks, assigns agents
3. **Agents** work in parallel, share context via Memory Bus
4. **RPA Agent** takes over UI when APIs unavailable
5. **Clerk Agent** writes code, tests, iterates autonomously
6. **Memory Engine** stores lessons, updates skills
7. **Results** aggregated and returned to user

## Design Principles

- **Zero Blocking**: All agents async, non-blocking
- **Graceful Degradation**: API down вҶ’ try RPA вҶ’ try search вҶ’ ask user
- **Self-Healing**: Auto-retry, auto-fix, auto-learn
- **Safety First**: Sandboxed execution, human confirmation for destructive ops
- **Hardware Optimized**: RTX 4080 = local vLLM + large models + fast inference
